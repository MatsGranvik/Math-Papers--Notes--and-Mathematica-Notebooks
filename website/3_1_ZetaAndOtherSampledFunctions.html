Visual Intuition about the Riemann Zeta Function
3-1: The Prime Number Theorem


Back to Part 1!



Okay, we've examined, extensively, a sampling error visually coinciding with the Riemann Zeta function.
And we've noted special amplitude/frequency pairs where the sampling error disappears, which are the fabled non-trivial Riemann Zeta Zeros. All the ones we looked at had an amplitude $A=-\frac{1}{2}$.
And we've even seen that we can express the Riemann Zeta function as a product of those zeros.
Thus far, we've seen that the sampling error is interesting and unpredictable, so maybe I've aroused your curiousity about it. Nevertheless, I don't think anything we've seen so far suggests why people care so very much about it (other than the cash prizes, of course).
My aim in this section is to change that. Let's finally broach the subject of prime numbers.
But there's one last concept we need to lead into broaching prime numbers.
Over the arc of these articles, I've repeatedly compared the sum $\sum_{t=1}^x t^{A+f i}$ to the integral $\int_{0}^x t^{A+f i} dt$. I've encouraged thinking of their difference as a sampling error.
Let's get more explicit with this sampling idea.
With just a touch of parameterization, we can unify both expressions as a single one. Suppose we introduce a sampling parameter $d$ and write the following equation:
$$F_d(x) = d \cdot \sum_{t=1}^{\lfloor \frac{x}{d} \rfloor } (d \cdot t)^{A+f i}$$
If $d=1$, this immediately yields our original sum as
$$F_1(x) = \sum_{t=1}^x t^{A+f i}$$
Meanwhile, if we interpret $d=0$ as a limit and notice that $F_d(x)$ is just a Riemann Sum, we get our integral as
$$F_0(x) = \int_{0}^x t^{A+f i} dt$$
So $F_d(x)$ generalizes both expressions. In fact, going down this road immediately raises questions about sampling errors at other values of $d$, such as $d=\frac{1}{2}$. I won't cover it here, but it's not much work to find that any positive real value of $d$ has the same non-trivial zeros, which is pretty striking.
Anyway, really take a second to ensure you follow this, becuase we're going to push this idea pretty hard in just a second.
Now let's rewrite $F_d(x)$ in a more compact form, which will seem arbitrarily complicated in the short term... but I have my reasons! We can remove the sum in $F_d(x)$ and rewrite the idea in a purely recursive form as
$$f_d(x,t) = f_d(x,t+d) + d \cdot ( t^{A+f i} )$$
with the terminating condition
$$ \text{if } t > x, f_d(x,t) = 0$$
With this, our original sum and integral are expressed as, respectively,
$$\sum_{t=1}^x t^{A+f i} = f_1(x,1)$$
and (interpretting $d=0$ as a limit)
$$\int_{0}^x t^{A+f i} dt = f_0(x,0)$$
Again, make sure you follow this.
Consequently, if $-1 < A < 0$, we can express the zeta function with $f(x,t)$ as this limit:
$$\zeta(-(A+f i)) = \lim_{x \to \infty} f_1(x,1) - f_0(x,0)$$
Up to this point, this is just restatements of familiar ideas with a new coat of notation.
But now let's use this notation to do something genuinely different. Let's take the general recursive form we just wrote for $f(x,t)$,
$$f(x,t) = f(x,t+d) + d \cdot ( \text{ stuff } )$$
with the terminating condition
$$ \text{if } t > x, f(x,t) = 0$$
and put other terms in that last parenthesis, where I've suggestively written "stuff".
I think you're going to be pretty surprised by what happens when we do.
So here are two variations to examine (with $f(x,t)$ for reference first):
$$f(x,t) = \color{gray}{f(x,t+d) + d \cdot } ( \bold{t^{A+f i}} )$$ $$g_k(x,t) = \color{gray}{g_k(x,t+d) + d \cdot } ( \bold{\frac{1}{k} - g_{k+1}(\frac{x}{t}, 1+d )} )$$ $$h(x,t) = \color{gray}{h(x,t+d) + d \cdot } ( \bold{ \log t - h(\frac{x}{t}, 1+d )} )$$
All three have the same terminating condition,
$$ \text{ If } t > x, \text{ then } f(x,t) = g_k(x,t) = h(x,t) = 0$$
Really look at these definitions and notice how they differ. Don't skip this! Really make sure you see how they are structurally similar, and also how they aren't.
The main difference, of course, is that $g_k(x,t)$ and $h(x,t)$ have a second recursive term, which is no small thing.
For what follows, let's define extra functions to set initial conditions for these recursive functions. I'll use uppercase letters to show which function goes with which. Those functions are
$$F_d(x,A+f i) = f(x,d)$$ $$G_d(x) = g_1(x,1+d)$$ $$H_d(x) = h(x,1+d)$$
So these functions set initial values for $t$. $g_k$ also starts with $k=1$. That's all that's happening here. There's nothing tricky going on.
In case you don't have a good head for reasoning about very compact recursive functions like $g_k$ when applied repeatedly to themselves, let's look at $F$, $G$, and $H$ in a more spawling form, which is actually immediately implied by these compact previous definitions.
If $d=1$, our three functions resolve to these sums:
$$F_1(x,A+f i) = \sum_{t=1}^{\lfloor x \rfloor} t^{A+f i}$$ $$G_1(x) = \sum_{t=2}^{\lfloor x \rfloor} 1 - \frac{1}{2}\sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor} 1 + \frac{1}{3}\sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor}\sum_{v=2}^{\lfloor \frac{x}{t \cdot u} \rfloor} 1 - ...$$ $$H_1(x) = \sum_{t=2}^{\lfloor x \rfloor} \log t - \sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor} \log t + \sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor}\sum_{v=2}^{\lfloor \frac{x}{t \cdot u} \rfloor} \log t - ...$$
And if $d=0$ (as a limit), we likewise are left with these integrals:
$$F_0(x,A+f i) = \int_0^x t^{A+f i}\,dt$$ $$G_0(x) = \int_1^x \,dt - \frac{1}{2}\int_1^x \int_1^\frac{x}{t} \,du \,dt + \frac{1}{3} \int_1^x \int_1^\frac{x}{t} \int_1^\frac{x}{t \cdot u} \,dv \,du \,dt - ... $$ $$H_0(x) = \int_1^x \log t \,dt - \int_1^x \int_1^\frac{x}{t} \log t \,du \,dt + \int_1^x \int_1^\frac{x}{t} \int_1^\frac{x}{t \cdot u} \log t \,dv \,du \,dt - ...$$
It's fair to say, in this form, that $G$ and $H$ look much more complicated than $F$, which I'll readily concede.
Nevertheless, if you compare $G_1(x)$ to $G_0(x)$, and $H_1(x)$ to $H_0(x)$, I hope you agree they do have a striking resemblance to the problem of the difference between $F_1(x, A+f i)$ and $F_0(x, A+f i)$. They can all be viewed fundamentally as discrete sampling error problems.
So now let's turn to a seemingly simple question.
We've already spent a lot of digital ink looking at $\sum_{t=1}^x t^{A+f i}$ and $\int_{0}^x t^{A+f i} dt$, renamed here as $F_1(x,A+f i)$ and $ F_0(x,A+f i)$. Notably, if $-1 < A < 0$ and $x$ is sufficiently large, then
$$F_1(x,A+f i) - F_0(x,A+f i) \approx \zeta(-(A+f i))$$
the Riemann Zeta function. So, a question - what does a similar difference look like for $G$ and $H$? What is the sampling error like for
$$G_1(x) - G_0(x) = ???$$
and for
$$H_1(x) - H_0(x) = ???$$
Okay - I think I'm dragging this out too long and playing far too coy.
It turns out, as we'll see visually in a second, $G_1(x)$ and $H_1(x)$ are two major prime number counting functions. And $G_0(x)$ and $H_0(x)$ are nice smooth approximations of those functions... all exactly as we've seen with $F_1(x,A+f i)$ and $F_0(x,A+f i)$.
So that's interesting.
Proving $G_1(x)$ and $G_0(x)$ (or the $H$ equivalents) can't differ by "too much" is essentially what is called the Prime Number Theorem, the name of which might suggest to you its importance.
In Riemann's original paper on what we now know as the Riemann Zeta function, its zeros were originally introduced for use in a formula explicitly describing the difference between $G_1(x)$ and $G_0(x)$. So, zeta zeros can describe the difference between this prime number counting function and its smooth approximation. $H_1(x)$ and $H_0(x)$ have a comparable formula using zeta zeros.
A specific, immediate consequence of the Riemann Hypthoesis being proven true (in the notation of the paper here, of all non-trivial zeros having $A=-\frac{1}{2}$) is that the difference between $G_1(x)$ and $G_0(x)$ is as small as it can possibly be (and likewise for $H_1(x)$ and $H_0(x)$).
So now let's look at these three sets of functions.

This article started by unifying $\sum_{t=1}^x t^{A+f i}$ and $\int_{0}^x t^{A+f i} dt$ with the recursive function $f(x,t)$. It then defined two other, structurally similar functions, $g_k(x)$ and $h(x)$, which turned out compute both prime number counting functions or their smooth approximations.
I chose these definitions to keep the descriptions of $g_k(x)$ and $h(x)$ as simple as possible (though by no means simple, still).
But coming up, we'll turn to something not, at all, very simple - explicit formulas for $G_1(x)-G_0(x)$, and for $H_1(x) - H_0(x)$, both expressed in terms of the non-trivial zeros of the zeta function (which are, remember, all values of $A+f i$ such that $\lim_{x \to \infty} F_1(x, A+ fi ) - F_0(x, A+ fi ) = 0$, if $-1 < A < 0$). This is one of the main reasons the Riemann Hypothesis is important in the first place.
We can define this richer, unified function, $D_z(x,s,d)$, as
$$f_k(x,t) = \color{gray}{f_k(x,t+d) + d }\cdot \bold{t^{s} \cdot (\frac{1+z}{k}-1) \cdot ( 1 + f_{k+1}(\frac{x}{t}, 1+d ) )}$$ $$ \text{ If } t > x, \text{ then } f_k(x,t) = 0$$ $$\bold{D_z(x,s, d)} = 1+f_1(x,1+d)$$
$$\Delta_z(x,s) = D_z(x,s,1)-D_z(x,s,0)$$
With this function, we can say that
$$F_1(x,s)-F_0(x,s) = \frac{s}{1+s}+\Delta_1(x,s)$$ $$G_1(x)-G_0(x) = \lim_{ z \to 0 } \frac{\partial}{\partial z} \Delta_z(x,0)$$ $$H_1(x)-H_0(x) = \lim_{ s \to 0 } \frac{\partial}{\partial s}\lim_{ z \to 0 } \frac{\partial}{\partial z} \Delta_z(x,s)$$
And remainders?
$$-q \cdot \Delta_1(\frac{p}{q},0)$$
$$D_z(x,s,1) = 1+\binom{z}{1} \cdot \sum_{t=2}^{\lfloor x \rfloor} t^{s} + \binom{z}{2} \cdot \sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor} (t \cdot u)^{s} + \binom{z}{3} \cdot \sum_{t=2}^{\lfloor x \rfloor}\sum_{u=2}^{\lfloor \frac{x}{t} \rfloor}\sum_{v=2}^{\lfloor \frac{x}{t \cdot u} \rfloor} (t \cdot u \cdot v)^{s} + ...$$ $$D_z(x,s,0) = 1+\binom{z}{1} \cdot \int_1^x t^{s} \,dt + \binom{z}{2} \cdot \int_1^x \int_1^\frac{x}{t} (t \cdot u)^{s} \,du \,dt + \binom{z}{3} \cdot \int_1^x \int_1^\frac{x}{t} \int_1^\frac{x}{t \cdot u} (t \cdot u \cdot v)^{s} \,dv \,du \,dt + ...$$
