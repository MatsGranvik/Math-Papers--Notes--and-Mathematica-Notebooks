Let's talk a little more about when the sine function equals zero, and why someone might care. This might seem insultingly basic, but I'm going to make a truly incredible jump in difficulty in just a second. This preliminary stuff will, I hope, give a point of reference for the more difficult stuff that follows.
So, let's look at what happens when we add a constant inside the sine function.
First, if we add a real number inside our sine function, its period stays the same but the curve shifts around. So, $\sin( z + 4 )$ looks like this:

$$y=\sin( x + 4 )$$

So now I've marked lines at $x=2 \pi - 4$, $x=4 \pi- 4$, $x=6 \pi- 4$, and so on. Once again, the interval from one line to the next marks an entire cycle, and $ \sin(x+4)=0$ when it intersects each line.
Alright. Another variation. We tried adding 4. Now let's add an imaginary number, $4 i $, as $ \sin( x + 4 i )$.

$$y=\sin(x + 4 i )$$

Click and drag graph to recenter. Use mousewheel to zoom in and out (or pinch on mobile). Graph code from here.
The big change, of course, is that now the function has both real and imaginary parts. The real part is blue, the imaginary is red. Right off the bat you might guess this makes it harder for the sine function to equal zero, since both the blue and red lines would need to intersect the x-axis at the same time.
But both the blue and red waves have the same wavelength - they're just out of phase. Thus, though they both pass through 0, they can't both pass through 0 at the same time. So $\sin( x + 4 i ) \ne 0$, for any real value of x.
And in fact, it turns out that the sine function is never equal to zero for any input with an imaginary component.
Okay - now let's move on. One last observation. The sine function can also be expressed as a Maclaurin series - just a fancy way of saying it can be expressed in terms of its various derivatives at $x=0$. It is
$$\sin(z) = \sin(0) + \sin'(0)\cdot z +\frac{\sin'(0)}{2!}\cdot z^2 +\frac{\sin^{(3)}(0)}{3!}\cdot z^3+\frac{\sin^{(4)}(0)}{4!}\cdot z^4+...$$
...which is...
$$\sin(z) = z -\frac{1}{6}\cdot z^3+\frac{1}{120}\cdot z^5-\frac{1}{5040}\cdot z^7-...$$
...or, more compactly,
$$\sin(z) = \sum_{k=0}^{\infty} (-1)^{k} \cdot \frac{z^{2k+1}}{(2 k + 1)!}$$
I hope this is familiar, but take a good, long look at this last equation. I really like visual symmetry and visual analogy, and about to call back to this.

And Now, A Brief Aside About Remainders

As simple as the sine function is, it's not the first function you encounter in math that behaves in a cyclical fashion.
That disinction goes, of course, to the remainder function.
I want to look, just for a second, at the remainder function with fresh eyes.
Suppose I define the following function, which takes real, positive values $x$ and $t$.
$$ f(x,d) = \sum_{k = 1}^{\lfloor \frac{x}{d} \rfloor } d$$
This defininitely isn't anything too tricky. There are two obversations I want to make about $f(x,d)$.
First, conceptually, what does $f(x,d)$ do? Well, think of it like this. Take a line segment $x$ units long. Remove as many non-overlapping line segments of length $d$ as possible, and sum up their length. That's what this function calculates. If $d$ divides $x$ evenly, that length will be $x$. If not, the total length will be less than $x$ but greater than $x-d$.
The second observation is that the limit here as $d$ approaches 0 is $ \displaystyle \lim_{d \to 0}{} \sum_{n = 1}^{\lfloor \frac{x}{d} \rfloor } d = \int_0^x dt$.
Let's combine these two observations. If we notate the remainder of x divided by t as $R(x,d)$, then
$$ R(x,d) = \int_0^x dt - \sum_{k = 1}^{\lfloor \frac{x}{d} \rfloor } d $$
And I suggest we think of $R(x,d)$, conceptually, as being a kind of sampling error.
The function looks like this (let's use $d=7$ as an example, so we'll be looking at $R(x,7)$):

$$y= \int_0^x dt - \sum_{k = 1}^{\lfloor \frac{x}{7} \rfloor } 7 $$

So now I've marked lines at $x=7$, $x=2 \cdot 7$, $x=3 \cdot 7$, and so on. Once again, the interval from one line to the next marks an entire cycle, and $ R(x,7)=0$ when it intersects each line. Of course, this probably seems like a lot of hullabaloo for the remainders you were taught in 3rd grade.
Nevertheless, let me bludgeon you over the head with my point here. When we look at the difference between sampling some line segment $x$ with infinitely small samples (that's our integral $\int_0^x dt$) versus sampling it with samples of size $d$ (that our sum $\sum_{k = 1}^{\lfloor \frac{x}{d} \rfloor } d$), the resulting sampling error has periodic properties as $x$ increases in size. It even takes on the value of 0 at the start of its cycle. Which is to say, here, there's some connections between sampling errors and periodicity.
Actually, this function goes by another name, a sawtooth wave. Tying back with the first topic of this article, it can be expressed very tidily as a sum of sine waves, a Fourier series, as $\displaystyle R(x,d) = d \cdot( \frac{1}{2} - \frac{1}{\pi} \cdot \sum_{k=1}^{\infty} \frac{1}{k} \sin(\frac{2 \pi k x}{d}) )$. Let's revisit our example of $R(x,7)$ by looking at an approximation of its Fourier series.

$$y= 7 \cdot( \frac{1}{2} - \frac{1}{\pi} \cdot \sum_{k=1}^{30} \frac{1}{k} \sin(\frac{2 \pi k x}{7}) ) $$
